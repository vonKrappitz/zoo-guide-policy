# Polityka Prywatności

**Ostatnia aktualizacja:** Luty 2026

## 1. Wstęp
Twórca aplikacji **Zoo Guide** szanuje Twoją prywatność. Ta polityka wyjaśnia, w jaki sposób traktujemy informacje podczas korzystania z naszej aplikacji mobilnej.

## 2. Gromadzenie danych
Aplikacja **Zoo Guide nie gromadzi, nie przechowuje, ani nie wysyła na zewnętrzne serwery żadnych danych osobowych**. Nie wymagamy zakładania konta, podawania e-maila ani logowania.

## 3. Dane zapisywane lokalnie (Local Storage)
Aby aplikacja działała poprawnie, zapisujemy pewne dane wyłącznie w pamięci Twojego urządzenia (tzw. Local Storage). Są to:
* Wybrany język i preferowane ZOO.
* Status zgody na korzystanie z aplikacji (potwierdzenie wieku).
* Ustawione lokalnie alarmy (np. przypomnienie o karmieniu lub parkowaniu).

Te dane nigdy nie opuszczają Twojego telefonu i możesz je w każdej chwili usunąć, czyszcząc dane aplikacji w ustawieniach systemu operacyjnego.

## 4. Uprawnienia urządzenia
Aplikacja korzysta z lokalnych powiadomień systemowych, aby przypomnieć Ci o zbliżającym się karmieniu zwierząt lub końcu czasu parkowania. Nie śledzimy Twojej lokalizacji (GPS).

## 5. Zmiany w Polityce Prywatności
Zastrzegamy sobie prawo do aktualizacji tej polityki. Wszelkie zmiany będą publikowane na tej stronie.
